Release Notes

v0.1 Oct 16, 2020

- Initial release as "KLMS VOD Controller".
- Binds shortcut keys to control KLMS VODs.

v0.2 Oct 28, 2020

- Added shortcut to control playback speed.

v2.0 Mar 2, 2021

- Updated to provide new features as well as to correspond to the new KLMS.
- New features include color themes, dark mode, icon vectorization, buttons to download lecture materials, automatic redirect to log-in page, and long VOD jump.

v2.1 Mar 20, 2021

- Colored, aligned & replaced some missed items.
- More features: options to set week to ALL when entering lecture rooms / scroll to current week when ↵ pressed / hide red circle when zero notifications.

v2.1.1 Apr 2, 2021

- Colored, aligned & replaced some missed items.
- Bug fix: Long jump is no longer activated permanently after switching windows with Alt/⌥ pressed.
