# KLMS VOD Controller

KLMS VOD 서비스에 단축키 기능을 부여하는 크롬 확장프로그램의 소스코드입니다. 단축키는 유튜브와 같은 키 바인딩을 사용합니다.

- 방향키: 전후 10초 탐색 (좌우) / 음량 10% 증감 (상하)

* M: 음소거 설정/해제

* F: 전체화면 설정/해제

* Z/X/C: 재생속도 1 / +0.2 / -0.2

* 스페이스바: 재생/일시정지

* 0~9: 동영상을 10분할해서 해당 숫자 지점으로 이동
